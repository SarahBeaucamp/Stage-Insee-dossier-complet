# Stage-Insee-dossier-complet
Ce dépôt Git contient le code R permettant de générer une page web interactive Observable dont l'objectif est de proposer des pistes de modernisation pour la représentation des données du dossier complet de l'INSEE.
Cette interface s'appuie sur dix fichiers au format qmd. Neuf d'entre eux explorent les différents thèmes et sous-thèmes du dossier complet tandis que le dernier est dédié aux chiffres clés du territoire de notre choix. En effet, l'interface web nous permet de sélectionner le territoire de notre choix et de naviguer de manière fluide entre tous les chiffres que l'on a concernant ce territoire.
Pour visualiser le projet, il faut se rendre dans le dossier nommé "total" qui regroupe tout ces fichiers qmd. Si les versions html de ces fichiers s'y trouvent déjà, il suffit d'en ouvrir une dans le navigateur web pour accéder au site. Si ce n'est pas le cas, il faudra d'abord compiler l'ensemble des fichiers qmd pour générer les pages html correspondantes avant de pouvoir y accéder.
